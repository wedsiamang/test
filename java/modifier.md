# アクセス修飾子
#### private
- privateメソッド
  privateメソッドは、サブクラスでも、異なるクラスでも、アクセスすることはできない。
- privateローカル変数
  privateがついたローカル変数はreturnに使うとコンパイルエラーになる
- 具象クラス、抽象クラス共にprivateメソッドは定義が可能  

#### final
- メソッドにfinalを付与するとオーバーライドを禁止する。
- finalメンバ変数に対して、newインスタンスにつき、１度のみ初期化を行える。インスタンスが別の場合は、それぞれ初期化を行える

#### default
アクセス修飾子なし（デフォルト）は、継承関係の有無に関わらず同じパッケージ以外からはアクセスができない。

#### protected
- サブクラスからもアクセスできる
- protected は「同一パッケージ」か「サブクラスとして」だけ使える。
- 別パッケージでは、親型参照からは使えない。
