# var 型推論
Local-Variable Type Interface
基本的にvarはローカル変数に使用できる。
### varが使用できる
- ローカル変数の初期化時
- 拡張for文内のインデックス
- 通常for文内で宣言されたローカル変数
- try-with-resources文内のローカル変数
- 定数宣言(final)は、var より前に記述する必要がある

### varが使用できない
- メンバ変数には使用できない(メソッドの外)
- 初期値を必ず代入する必要がある
- メソッド、コンストラクタの引数には使用できない
- メソッド戻り値には使用できない
- catchブロックでは使用できない
- 右辺がnullの場合型推論ができないため使えない

### varは予約語ではない
予約語ではないため、識別子として使用可能
