# ポリフォーイズム
- mainからサブクラスを呼び出し、該当メソッドがない場合、super()でスーパークラスを探しに行き、メソッドがあり、そのメソッドがオーバーライドされていれば、オーバーライドのサブクラスを優先する。

#### 多態
多態（polymorphism）とは、同じメソッド呼び出しが、実行時のインスタンスの型によって別の実装に切り替わる性質
実行時多態では、実体クラスでオーバーライドされたメソッドが選ばれる
多態はメソッド呼び出しにのみ適用され、インスタンス変数はオーバーライドの対象ではないため、参照型(左辺の型）に基づいて解決される。





- スーパークラスで定義したstatic変数、インスタンス変数、staticメソッド、インスタンスメソッドをサブクラス側で再定義することは可能
- サブクラスのオブジェクトをスーパークラス型の変数に代入している場合、インスタンスメソッド以外はスーパークラスのメンバが呼び出される。
```
A obj = new B();
obj.method();
```
