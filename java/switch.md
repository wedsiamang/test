# switch
#### switchで使えるデータ型
char,byte,short,int,これらのラッパークラス、enum,String

### 頻出問題
##### 'c' | 'd' :
switchの | はビット演算で、どちらかが１のときのみ1を返し、それ以外は0を返す。
swtichの構文として問題ないためコンパイルエラーとはならない。

##### 1 | 2
 case 1 | 2: = 0001 | 0010 → 0011 = 3 結果：3

 #### "c" | "d" :
 ビット演算は整数型に対して行われるため、ダブルクオーテーションの文字型ではコンパイルエラーとなる。

#### コンパイルエラー
- case に使える変数は定数のみ
- switch(引数）と型が一致していないものはコンパイルエラーとなる
