# static
- static メソッドからインスタンスメンバへ直接アクセスはできない。
- static を付与したメソッドは、staticメソッドとなる。
- static main メソッドから、staticメンバへは、newインスタンスを作成しなくても呼び出しが可能、staticメソッドではなくインスタンスメソッドの場合は、newしなければ呼び出せない
- staticメソッドから、フィールドのインスタンスメソッドにアクセスすることはできない、コンパイルエラーになる。


#### インスタンス変数
- newしなければアクセスできない、Main.num1のようにクラス名、インスタンスメンバの呼び出しはできない、必ずnew Main().num1のようにインスタンス化してからアクセスする必要がある

#### メンバ間のアクセスルール
- クラス内に定義したインスタンスメンバ同士、staticメンバ同士は直接アクセスできる
- クラス内で定義したインスタンスメンバは、クラス内で定義したstaticメンバに直接アクセスできる
- クラス内で定義したstaticメンバは、クラス内で定義したインスタンスメンバに直接アクセスできない。インスタンス化したらアクセスできる。
- staticメソッドは、「クラス名.メソッド名」という呼び出しが許可されているが、インスタンスメンバは許可されていない。
